### 命名规范

>严格使用页面+功能区块名

导航栏
header：导航栏
header-logo：导航栏logo
header-nav：导航栏导航

文章列
post：文章列
post-item：文章列表项
post-link：文章跳转链接
post-title：文章标题
post-description：文章描述
post-published：文章发布时间

页脚
footer：页脚
footer-copyright：页脚版权
footer-nav：页脚导航

友链
link：友链
link-top：友链上区块
link-bottom：友链下区块
link-item：友链列表项
link-link：友链跳转链接
link-name：友链名称

日记
diary：日记
diary-item：日记列表项
diary-link：日记跳转链接
diary-title：日记标题
diary-content：日记内容
diary-published：日记发布时间
